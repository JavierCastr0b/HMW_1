# Funciones

empty() → verifica si la lista está vacía.

size() → retorna el tamaño de la lista.

push_front(x) → inserta un elemento al inicio.

push_back(x) → inserta un elemento al final.

T_front() → devuelve el primer elemento.

T_back() → devuelve el último elemento.

pop_front() → elimina y retorna el primer elemento.

pop_back() → elimina y retorna el último elemento.

operator[](i) → retorna el elemento en la posición i.

clear() → elimina todos los elementos.

reverse() → invierte la lista.

sort() → ordena la lista en orden ascendente.
